Heya! I would really like to thank you for using the PFP maker!

If you want to contribute art, please read the following:

» I have made the art in canva, click the button below to access it.

» please make sure that you create a seperate page to create your own designs, don't delete or modify existing ones.

» If you want to modify an existing design, duplicate the page and make the required changes.

» Check if your design aligns with the character's position, I would recommend to duplicate a page with the character in it (eg: char-2) and start designing on top of it, you can remove the character when you're done with the design.

» Then download only your designs, and remove the background using (remove.bg)

» Then download the images from remove.bg and send it to me through slack (@siddharthan)

» I will really appreciate such contributions! Please DO NOT misuse the link. Thanks :]
